Introduction

This task is a small project for you to demonstrate your ability to understand the requirements and implement a solution using Javascript in a fixed timeframe.

For this task, we’re asking you to build a list of some records with the ability to view, edit and delete individual records.

Project Goals

Render a list of records (Cats).

a list of records (Cats).  Upon clicking on a cat, a detailed view should appear with more details about the selected record.

Clicking on “edit” would open an overlay to edit the selected cat attributes.

Clicking on “delete” remove the selected cat from the list.
